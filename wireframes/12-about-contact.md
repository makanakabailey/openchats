## Key Components

### About Page
- **Hero Section**: Brand identity and tagline
- **Story Section**: Origin and mission
- **Host Profiles**: Photos, bios, social links
- **What We Do**: Three pillars (Podcast, Community, Merch)
- **Values**: Core principles
- **Statistics**: Key metrics and achievements
- **Featured In**: Media mentions and press
- **CTA**: Link to contact page

### Contact Page
- **Contact Form**: Multi-purpose inquiry form
- **Contact Methods**: Email, phone, social media
- **Business Hours**: SAST timezone
- **FAQ Section**: Common questions with expandable answers
- **Specific Inquiries**: Department-specific email addresses
- **Newsletter Signup**: Email subscription with preferences

---

## Form Fields

### Contact Form
- **Name** (required): Text input
- **Email** (required): Email input with validation
- **Subject** (required): Dropdown select
  - General Inquiry
  - Guest Appearance
  - Sponsorship
  - Technical Support
  - Feedback
  - Other
- **Message** (required): Textarea (min 50 characters)
- **Attachment** (optional): File upload (max 5MB)

### Newsletter Form
- **Email** (required): Email input with validation
- **Preferences** (optional): Checkboxes
  - Weekly episode updates
  - Exclusive content
  - Merchandise drops
  - Community events

## Error Handling

### Form Validation Errors
- Empty required fields: "This field is required"
- Invalid email: "Please enter a valid email address"
- Message too short: "Message must be at least 50 characters"
- File too large: "File size must be under 5MB"
- Upload failed: "File upload failed. Please try again"

### Network Errors
- Connection failed: "Unable to send message. Please check your connection and try again"
- Server error: "Something went wrong. Please try again later or email us directly at hello@openchats.co.za"
