## Key Components

- **Tab Navigation**: Episode ideas, merch ideas, questions
- **Submission Form**: Name, email, topic, description
- **File Upload**: Attach supporting materials
- **Recent Submissions**: Community voting on ideas
- **Vote System**: Upvote popular submissions

## Interactions

- Switch between submission types via tabs
- Optional authentication for submissions
- Vote on community submissions
- Filter by most voted/recent
- Hosts can mark submissions as "Under Review" or "Implemented"
