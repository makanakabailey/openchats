## Key Components

- **Image Gallery**: Main image with thumbnail carousel
- **Product Info**: Title, price, rating, description
- **Variant Selectors**: Size, color, style options
- **Quantity Selector**: +/- buttons
- **CTA Buttons**: Add to cart (primary), wishlist (secondary)
- **Trust Badges**: Shipping info, POD timeline
- **Tabs**: Description, size guide, reviews, shipping
- **Related Products**: Cross-sell recommendations

## Interactions

- Click thumbnail to change main image
- Zoom on hover/click main image
- Variant selection updates price/availability
- Add to cart shows confirmation toast
- Reviews section with filtering and sorting
