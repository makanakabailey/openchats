## Key Components

- **Live Indicator**: Red dot, viewer count, duration
- **Chat Window**: Real-time scrolling messages
- **Online Users**: Sidebar showing participants
- **Host Badge**: Special indicator for host messages
- **Pinned Messages**: Important announcements stay at top
- **Message Input**: Text field with emoji/image support
- **Upcoming Events**: Calendar of scheduled live sessions
- **Countdown Timer**: Shows time until next event
- **Past Recordings**: Archive of previous live chats

## Interactions

- Messages auto-scroll (with pause option)
- Click username to view profile
- Moderators can pin/delete messages
- Emoji reactions to messages
- Set reminders for upcoming events
- Mobile: Full-screen chat mode
