## Key Components

### Cart
- Item list with images, variants, quantity controls
- Remove item button
- Order summary sidebar
- Discount code input
- Progress indicator for free shipping threshold

### Checkout
- Multi-step form (contact, shipping, payment)
- Order summary (sticky on scroll)
- Payment gateway integration (Stripe/PayPal)
- Security badges
- Mobile: Single column, summary collapsible
