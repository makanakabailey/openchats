## Key Components

- **Live Event Banner**: Highlights active live chats/Q&As
- **Filter Tabs**: Hot, Recent, Top, Unanswered
- **Category Sidebar**: Themed discussion areas with post counts
- **Discussion List**: Thread title, author, timestamp, reply count
- **Active Members**: Shows who's currently online
- **New Discussion Button**: Prominent CTA to start conversations

## Interactions

- Click category filters discussion list
- Hover shows thread preview
- Pin/feature important threads
- Real-time updates for new posts
- Badge indicators for unread threads
