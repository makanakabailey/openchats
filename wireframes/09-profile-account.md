## Key Components

- **Profile Header**: Avatar, username, join date, badges
- **Tabs**: Activity, orders, favorites, settings
- **Stats Sidebar**: Post count, likes, purchases
- **Activity Feed**: Chronological user actions
- **Order History**: Past purchases with tracking
- **Settings**: Profile, notifications, privacy controls

## Interactions

- Edit profile inline
- Filter activity by type
- Reorder past purchases
- Export order history
- Manage notification preferences
