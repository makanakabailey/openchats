## Design Specifications

### Colors
```
Background: #000000 (black) or #09090b (zinc-950)
Cards: #18181b (zinc-900)
Text Body: #d4d4d8 (zinc-300)
Text Muted: #a1a1aa (zinc-400)
Accent: #FF4500 (Signal Orange)
Borders: #27272a (zinc-800)
```

### Typography
```
Display Font: Clash Display, Oswald
Body Font: DM Sans, Inter

Hero: 72px, Font-weight 900, UPPERCASE
H1: 60px, Font-weight 900, UPPERCASE
H2: 48px, Font-weight 700, UPPERCASE
```

### Components
```
Primary Button:
- Background: #FF4500 (Orange)
- Text: White
- Font: Bold, UPPERCASE
- Hover: Scale 1.05x, Glow effect

Secondary Button:
- Border: 2px White
- Background: Black
- Text: White
- Font: Bold, UPPERCASE
- Hover: Invert colors

Card:
- Background: zinc-900
- Border Radius: 0.5rem
- Hover: Scale 1.05x, Orange glow
- Transition: 0.3s ease

Badge:
- Background: Orange
- Text: White, Bold, UPPERCASE
- Padding: 0.5rem 1rem
- Border Radius: 9999px (full)
```

### Effects
```
Glow: 0 0 30px rgba(255, 69, 0, 0.5)
Glassmorphism: backdrop-blur(12px)
Hover Scale: transform: scale(1.05)
Transition: 0.3s ease
```
